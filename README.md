## üöÄ Getting Started

### Installation

#### On Ubuntu/Debian

```bash
# Update package list
sudo apt update

# Install Python 3
sudo apt install python3

# Install pip (Python package manager)
sudo apt install python3-pip

# Verify installation
python3 --version
pip3 --version
```
